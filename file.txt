When running Jenkins on a GKE (Google Kubernetes Engine) cluster, you need certain plugins to ensure smooth operation, integration with Kubernetes, and enhanced CI/CD workflows. Below is a categorized list of necessary Jenkins plugins for GKE:
Core Plugins

These are essential plugins for Jenkins functionality:

    Kubernetes Plugin
        ID: kubernetes
        Purpose: Integrates Jenkins with Kubernetes for dynamic agent provisioning.
        Install Command:

    - kubernetes:3978.vb_4ea_388cf528

Pipeline Plugins

    Workflow API (workflow-api): Core plugin for pipeline execution.
    Workflow Aggregator (workflow-aggregator): Installs pipeline functionality.
    Pipeline Stage View (pipeline-stage-view): Displays pipeline execution stages.
    Install Command:

    - workflow-api:2.47
    - workflow-aggregator:2.6
    - pipeline-stage-view:2.19

SSH Credentials Plugin

    ID: ssh-credentials
    Purpose: Manage SSH credentials for Jenkins.
    Install Command:

        - ssh-credentials:273.v85f8623c1c5a

Google Cloud Plugins

For seamless integration with Google Cloud services:

    Google Kubernetes Engine Plugin
        ID: google-kubernetes-engine
        Purpose: Supports GKE-specific functionality.
        Install Command:

    - google-kubernetes-engine:0.9.0

Google OAuth Credentials

    ID: google-oauth-plugin
    Purpose: Authenticate Jenkins to access Google Cloud APIs.
    Install Command:

    - google-oauth-plugin:1.0.0

Google Cloud Storage Plugin

    ID: google-cloud-storage-plugin
    Purpose: Upload/download artifacts to Google Cloud Storage.
    Install Command:

        - google-cloud-storage-plugin:1.4

Kubernetes and Container Plugins

For efficient Kubernetes integration and containerized workflows:

    Kubernetes CLI Plugin
        ID: kubernetes-cli
        Purpose: Execute kubectl commands from Jenkins.
        Install Command:

    - kubernetes-cli:1.13.0

Docker Pipeline Plugin

    ID: docker-workflow
    Purpose: Enables Jenkins to build and deploy Docker containers.
    Install Command:

    - docker-workflow:1.28

Kubernetes Credentials Plugin

    ID: kubernetes-credentials-provider
    Purpose: Use Kubernetes secrets as Jenkins credentials.
    Install Command:

        - kubernetes-credentials-provider:0.20

Build and Deployment Plugins

For automating builds and deployments:

    Git Plugin
        ID: git
        Purpose: Interact with Git repositories.
        Install Command:

    - git:5.0.0

GitHub Plugin

    ID: github
    Purpose: Integrate Jenkins with GitHub repositories.
    Install Command:

    - github:1.37.0

Artifact Uploaders

    S3 Plugin (s3): For AWS S3.
    Google Cloud Storage Plugin: For GCS.
    Install Command:

        - s3:0.11.5

Security and Credentials Plugins

For managing secure connections and credentials:

    Credentials Binding Plugin
        ID: credentials-binding
        Purpose: Inject credentials into jobs.
        Install Command:

    - credentials-binding:1.27

Role-Based Authorization Plugin

    ID: role-strategy
    Purpose: Fine-grained role-based access control.
    Install Command:

    - role-strategy:3.3

Matrix Authorization Strategy Plugin

    ID: matrix-auth
    Purpose: Define permissions for multiple users/groups.
    Install Command:

        - matrix-auth:3.1.7

Monitoring and Optimization Plugins

For better monitoring and resource usage:

    Metrics Plugin
        ID: metrics
        Purpose: Collect system metrics for Jenkins.
        Install Command:

    - metrics:4.0.2

Performance Plugin

    ID: performance
    Purpose: Analyze performance of builds.
    Install Command:

    - performance:3.20

Monitoring Plugin

    ID: monitoring
    Purpose: Monitor Jenkins nodes and jobs.
    Install Command:

        - monitoring:1.92.0

UI and Quality of Life Plugins

For better user experience:

    Blue Ocean Plugin
        ID: blueocean
        Purpose: Provides a modern pipeline visualization UI.
        Install Command:

    - blueocean:1.26.0

AnsiColor Plugin

    ID: ansicolor
    Purpose: Adds color to console output for better readability.
    Install Command:

        - ansicolor:1.0.1

Install and Validate Plugins

    Deploy Jenkins:

helm upgrade --install jenkins jenkinsci/jenkins -f values.yaml -n new-jenkins3

Check installed plugins:

    kubectl logs jenkins-0 -n new-jenkins3

    Verify in Jenkins UI under Manage Jenkins > Plugin Manager.
